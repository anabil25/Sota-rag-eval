793-6          [EIS](#)Eligibility Information System INFORMATION

|  |  |  |
| --- | --- | --- |
|  | [MFU](#)Mandatory Filing Unit Member | Non-[MFU](#)Mandatory Filing Unit Member |
| [WOSA](#)Work Sanctions (Eligibility Information System Screen) Code | IP | IP |
| [SEPA](#)Setup Participants (Eligibility Information System Screen) Code | DF | OU |

If the disqualified individual is required to be included in the assistance unit, the resources and income of that individual are still counted in the eligibility and payment determinations for the assistance unit.

If the disqualified individual is not required to be included in the assistance unit, that individual must be removed from the assistance unit. The resources and income of that individual are not counted in the eligibility and payment determinations for the remaining assistance unit.
