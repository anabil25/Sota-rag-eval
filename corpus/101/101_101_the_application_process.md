101            THE APPLICATION PROCESS

In order for the Division of Public Assistance (DPADivision of Public Assistance) to determine an individual's eligibility for public assistance benefits, the individual must:

- Submit an identifiable application;
- Complete and sign an approved application form;
- Attend an interview with a DPADivision of Public Assistance caseworker, fee agent, or Native Family Assistance Program (NFAPNative Family Assistance Program) agency staff person, if an interview is required; and,
- Provide documentation and verification, including required forms, needed to determine program eligibility.

This chapter provides basic information on the application process. Refer to the appropriate program manual for specific program rules.
