Work Services Procedure Manual

[Click here to see this page in full context](#)

STATE OF ALASKA 

DEPARTMENT OF HEALTH

Division of Public Assistance

TRANSMITTAL NUMBER:  MC #17

MANUAL:  Work Services Procedure Manual

DATE:  December 1, 2025

We have made technical corrections and incorporated the changes outlined below.

If you have any questions, please contact the Policy and Program Development team at [hss.dpa.policy@alaska.gov](mailto:hss.dpa.policy@alaska.gov)

OVERVIEW OF CHANGES

ENTIRE MANUAL

Replaced reference to [hss.dpa.office@alaska.gov](mailto:hss.dpa.office@alaska.gov) with direction to use the [DPA's Secure Document Portal](https://dpaforms.ilinx.com/ILINXFlex/public-eform?param=WEUwT0U5d0g5QTV2aFZiZlI2Z0VEdyUzRCUzRCAg) to upload requests and documents.

[MS 1001-2](../1001/1001-2_employability_assessment.md) EMPLOYABILITY ASSESSMENT

- Removed the requirement to refer ineligible aliens to Work Services.

[MS 1001-13](../1001/1001-13_documentation.md) DOCUMENTATION

- Updated TA5 documentation requirements.

[MS 1002-1](../1002/1002-1_self-sufficiency_planning_requirements.md) SELF-SUFFICIENCY PLANNING REQUIREMENTS

- Removed the option for ineligible aliens to voluntarily participate in [FSSP](#)Family Self Sufficiency Plan development.

[MS 1003-1](../1003/1003-1_performance_measures.md) PERFORMANCE MEASURE DEFINITIONS

- Deleted bullets referencing ineligible aliens.

[MS 1003-3](../1003/1003-3_work_verification.md) WORK VERIFICATION CORE ACTIVITIES

- Added In-Kind work examples.
- Updated Paid Employment work verification requirements to align with eligibility requirements.
- Revised section for clarity.

[MS 1004-3](../1004/1004-3_what_supportive_services_cannot_be_used_for.md) WHAT SUPPORTIVE SERVICES CANNOT BE USED FOR

- Revised section for clarity.
- Expanded the list of basic services to be paid for with client's benefits.
- Revised parameters for paying phone/cell phone and internet services.

[MS 1006-6](../1006/1006-6_if_good_cause_does_not_exist.md) IF GOOD CAUSE DOES NOT EXIST

- Updates the reference from the HowWe guide to MS 1007-2.

[MS 1007-1](../1007/1007-1_penalty_types_and_their_impact.md) PENALTY TYPES AND THEIR IMPACT

- Clarified when an SS or WA penalty should be used.

[MS 1007-14](../1007/1007-14_disqualifications.md) DISQUALIFICATIONS

- Aligned language regarding Job Quit between [WSPM](#)Work Services Procedure Manual and [ATAP](#)Alaska Temporary Assistance Program MS 722-4.

[MS 1015-11](../1015/1015-11_quarterly_continuous_improvement_plan_(qcip).md) QUARTERLY CONTINUOUS IMPROVEMENT PLAN (QCIP)

- Corrected the link.

[MS 1018-9](../1018/1018-9_archiving_and_destruction_of_files.md) ARCHIVING FILES

- Revised file retention requirements.
