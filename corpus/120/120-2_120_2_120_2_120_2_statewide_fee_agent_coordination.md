120-2          STATEWIDE FEE AGENT COORDINATION

Coordination of the fee agent process is assigned to DOSTDivision Operations Support Team. This ensures consistency in implementing the process statewide. DOSTDivision Operations Support Team's responsibilities include:

- Accepting fee agent applications ([FA 50](#)Fee Agent Application);
- Recruiting and selecting fee agents;
- Conducting interview on selected candidate and reference checks;
- Maintaining and updating active list of fee agents;
- Collecting Monthly Fee Agent Billing Report ([FA 48](#)Collecting Monthly Fee Agent Billing Report) forms from fee agents;
- Verifying information from the Fee Agent Monthly Billing Report ([FA 48](#)Collecting Monthly Fee Agent Billing Report) was received;
- Submitting and uploading Fee Agent Billing Report ([FA 8](#)Fee Agent Billing Report) to Finance SharePoint;
- Developing and maintaining the Fee Agent Manual;
- Developing statewide fee agent forms;
- Developing training materials and guides;
- Managing the fee agent billing process;
- Providing assistance and clarification to field offices regarding fee agent-related questions, issues, and concerns;
- Verifying training completion and Fee Agent Provider Agreement signature;
- Signing the Fee Agent Provider Agreement ([FA 33](#)Fee Agent Provider Agreement) on behalf of DPADivision of Public Assistance; and
- Sending letters of termination and reminder letters related to Fee Agent Provider Agreement ([FA 33](#)Fee Agent Provider Agreement) and obligations to fee agents.

DPADivision of Public Assistance offices that use fee agents assign staff to work with fee agents. The local office responsibilities include:

- Accepting applications, reviews, and other documents and verification from fee agents; and
- Handling supply order form ([FA 9](#)Fee Agent Order Form) from fee agents.
