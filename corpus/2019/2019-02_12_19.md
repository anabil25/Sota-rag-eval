[Click here to see this page in full context](#)

**STATE OF ALASKA**

**Department of Health
and Social Services**

**Division of Public
Assistance**

**TRANSMITTAL NUMBER:**  2019-02

**MANUAL:**  Senior Benefit Program

**DATE:**  December 2, 2019

We have revised the Senior Benefits Program Manual
with updated Federal Poverty Levels for Alaska. Other changes are either
technical in nature or clarifications that are the result of policy questions
or suggestions from the field.

Highlights of these revisions are described below.
 If you have any questions please contact the Policy and Program
Development Team by e-mail at [dpapolicy@alaska.gov](mailto:dpapolicy@alaska.gov)

**OVERVIEW OF CHANGES**

[310-2A](../../310__application/310-2_actions_taken_on_the_application.md#310-2_A._Approving_the_Application)
     APPROVING APPLICATION

- Removed
  note requiring caseworkers to set an alert on any associated SNAP
  or TA case when a SB case is approved as this is no longer the process
  now that cases are handled statewide.

**[350](../../350__institutions/350_institutional_residency.md)           INSTITUTIONS**

- Added an individual is eligible for Senior
  Benefits while living in an Assisted Living Home.

[360-3](../../360__case_maintenance/360-3_suspension_and_termination.md)
     SUSPENSION AND TERMINATION

- Removed note
  requiring caseworkers to set an alert on any associated SNAP or TA
  case when a SB case is terminated as this is no longer the process
  now that cases are handled statewide.

[362](../../362__fair_hearings/362_fair_hearings.md)
        FAIR HEARINGS

- Updated
  section adding a link to [ADMIN MS 117-2](http://dpaweb.hss.state.ak.us/manuals/admin/117/117-2.htm) for
  fair hearing information.
