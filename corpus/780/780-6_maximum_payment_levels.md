780-6          MAXIMUM PAYMENTS LEVELS

The following tables show the maximum amount of [ATAP](#)Alaska Temporary Assistance Program benefits payable to a family of the appropriate type and size. These amounts represent the benefit amount payable to an assistance unit that has no countable income and receives the maximum shelter allowance.

The amounts in these tables are not used to calculate [ATAP](#)Alaska Temporary Assistance Program benefit payments; they are presented here for information purposes only.

780-6 A.      BASIC ADULT-INCLUDED PAYMENT MAXIMUMS

The maximum payment amount for cases that include one caretaker relative or two parents who are both able to perform gainful activity, and any number of dependent children, are:

|  |  |  |
| --- | --- | --- |
| Assistance Unit Size | | |
| One Caretaker Relative | Two-Parent Family | Maximum Payment |
| 2 | 3 | $821 |
| 3 | 4 | $923 |
| 4 | 5 | $1,025 |
| 5 | 6 | $1,127 |
| 6 | 7 | $1,229 |
| 7 | 8 | $1,331 |
| 8 | 9 | $1,433 |
| Each Additional Child |  | $102 |

780-6 B.      INCAPACITATED ADULT-INCLUDED PAYMENT MAXIMUMS

The maximum payment amount for cases that include two parents and one parent is physically or mentally unable to perform gainful activity, and any number of dependent children, are:

|  |  |
| --- | --- |
| Assistance Unit Size | Maximum Payment |
| 3 | $923 |
| 4 | $1,025 |
| 5 | $1,127 |
| 6 | $1,229 |
| 7 | $1,331 |
| 8 | $1,433 |
| 9 | $1,535 |
| Each Additional Child | $102 |

780-6 C.      CHILD-ONLY PAYMENT MAXIMUMS

The maximum payment amounts for cases that do not include a caretaker relative in the assistance unit are:

|  |  |
| --- | --- |
| Assistance Unit Size | Maximum Payment |
| 1 | $452 |
| 2 | $554 |
| 3 | $656 |
| 4 | $758 |
| 5 | $860 |
| 6 | $962 |
| 7 | $1,064 |
| 8 | $1,166 |
| Each Additional Child | $102 |

780-6 D.      PREGNANT WOMEN PAYMENT MAXIMUM

The maximum payment amount for a third trimester pregnant woman case is $514.00.
