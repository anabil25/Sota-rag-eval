723-7          [EIS](#)Eligibility Information System INFORMATION

This chart provides basic information on the different penalties.

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| Type of Penalty | [WOSA](#)Work Sanctions (Eligibility Information System Screen) Code | [SEPA](#)Setup Participants (Eligibility Information System Screen) Code | | Notice |
|  |  | [MFU](#)Mandatory Filing Unit Member | Non-[MFU](#)Mandatory Filing Unit Member |  |
| [CSED](#)Child Support Enforcement Division Non-Coop | CS | DI | OU | W709 |
| [FSSP](#)Family Self-Sufficiency Plan | SS | DJ | OU | W787 |
| Work Activities | WA | DJ | OU | W787 |
| Minor Parent School Attendance | SA | DI | OU | W713 |

If the penalized individual must be included in the assistance unit, the resources and income of that individual are still counted in the eligibility and payment determinations for the assistance unit.

If the penalized individual is not required to be included in the assistance unit, that individual is removed from the assistance unit, and their resources and income are not counted in the eligibility and payment determinations for the remaining assistance unit.
