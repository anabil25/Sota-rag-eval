Administrative Procedures Manual

[Click here to see this page in full context](#)

STATE OF ALASKA

Department of Health and Social Services

Division of Public Assistance

TRANSMITTAL NUMBER: 2010-02

MANUAL: Administrative Procedures Manual

DATE: December 3, 2010

We have revised several sections of the Administrative Procedures Manual in response to requests from staff. The changes include:

- Procedures for handling returned mail is now included in the manual which includes instructions on processing, documenting, and shredding the mail.
- Guidelines that clarify storage and shredding of request for service applications.
- Procedures for requesting case records from Archives.
- Actions to take on certain alerts.
- Actions to take in handling cases with undocumented aliens.

Highlights of these revisions are described below. If you have any questions, please contact the Policy and Program Development Team at 465-3382 or email [dpapolicy@alaska.gov](mailto:dpapolicy@alaska.gov).

OVERVIEW OF CHANGES

[MS 103-1](../103/103-1_electronic_document_management.md)      THE CASE FILE FOLDER

- Provides guidance instructing staff that returned mail will not be retained in the case file.
- Deletes all references suggesting that returned mail are kept in the file or transferred to a new file when voluming case files.
- Includes returned mail as one of the examples of documents staff may purge from case files.

[MS 103-2](../103/103-2_identifiable_applications.md)      REQUEST FOR SERVICE

- Provides guidance on when to pull requests out of the monthly file.
- Clarifies the procedure for storing request of service applications in the office (requests must be kept for three years).
- Clarifies when request for service applications can be destroyed or shredded.

[MS 103-4](../103/103-4_processing_returned_mail.md)      PROCESSING RETURNED MAIL

- Provides guidance on handling mail that is returned to DPA office for various reasons.

[MS 103-6](../103/103-6_retention_of_case_records.md)      RETENTION OF CASE RECORDS

- Provides instructions to field offices when requesting case records from Archives.

[MS 104-3](../104/104-3_tools_for_effective_workload_management.md)      TOOLS FOR EFFECTIVE CASE LOAD MANAGEMENT

- Includes guidance on processing alerts and the required action needed to ensure cases are worked properly.

[MS 104-4](../104/104-4_adverse_action_calendar.md)      ADVERSE ACTION CALENDAR

- Clarifies the 10-day count for sending adverse action notices begins the day after the date the notice is mailed.

[MS 105-14](../105/105-14_systematic_alien_verification_for_entitlements_-_electronic_verification_process.md)      SYSTEMATIC ALIEN VERIFICATION FOR ENTITLEMENT

- Includes guidance on handling cases with suspected undocumented aliens.

[MS 109-9](../109/109-9_additional_guidelines.md)      CASE NOTE DOCUMENTATION

- Provides guidance on documenting receipt of returned mail.

MS 127-5      CHILD NEEDS TRANSPORTATION TO AN IN-STATE TREATMENT FACILITY

- Revises the contact information for sending TEFRA Medicaid packet submitted by parents for children applying for TEFRA Medicaid

[ADDENDUM 103 A](../addenda/addendum_103_a_-_records_reference_request.md)

- Adds the Records Reference Request form
