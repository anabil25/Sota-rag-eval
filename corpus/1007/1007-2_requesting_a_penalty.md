1007-2          REQUESTING A PENALTY

The following process applies when a Work Services case manager has determined that a [WA](#)Work Activity or [SS](#)Supportive Services penalty is appropriate.

Case manager actions:

1. Enter a Client Note ([CLNO](#)Client Notes) and Case Note ([CANO](#)Case Note) documenting the penalty code and reason.
2. Send the request and supporting documentation to [DPA's Secure Document Portal](https://dpaforms.ilinx.com/ILINXFlex/public-eform?param=WEUwT0U5d0g5QTV2aFZiZlI2Z0VEdyUzRCUzRCAg).

Note:

Work eligible individuals (pregnant women, single parents, and/or two-parent household types) are optional for the purposes of receiving assistance and work activity participation during the period of retro-activity (i.e. time period between the application date and the approval date of benefits). Please refer to [WSPM](#)Work Services Procedure Manual [Section 1001-1](../1001/1001-1_rapid_engagement.md) for guidance regarding rapid engagement.  Penalty requests during the time of retro-activity are allowed but discouraged.
