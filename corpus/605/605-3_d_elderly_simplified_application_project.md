605-3 D.      ELDERLY SIMPLIFIED APPLICATION PROJECT

This section describes policies related to Alaska's Elderly Simplified Application Project (ESAP). [ESAP](#)Elderly Simplified Application Project Demonstration streamlines various requirements, such as verification and recertification, for eligible elderly/disabled households applying for or participating in [SNAP](#)Supplemental Nutrition Assistance Program.

Background

Department of Health, Division of Public Assistance, requested and received federal approval to implement an Elderly Simplified Application Project (ESAP) demonstration project. This demonstration has been approved to operate effective 12/01/2024.

1. Eligibility

   A household is eligible for [ESAP](#)Elderly Simplified Application Project Demonstration if it does not receive countable earned income and where all adult members are:

- Age 60 or older; and/or
- Have a disability;

Children may be present in the household, however the presence of a disabled child(ren) in the household alone will not make a household [ESAP](#)Elderly Simplified Application Project Demonstration eligible.

1. Application and Screening

   [ESAP](#)Elderly Simplified Application Project Demonstration households will complete the same application as non-ESAP households for initial applications and recertifications. 

   [ESAP](#)Elderly Simplified Application Project Demonstration households are eligible for Expedited Service. ESAP applications must be screened and processed to adhere to [SNAP](#)Supplemental Nutrition Assistance Program timeframes outlined in SNAP [MS 601](../601/601.md).

1. Interview

   At application, [ESAP](#)Elderly Simplified Application Project Demonstration households must complete an initial interview in order to determine eligibility for the ESAP demonstration project. This is done to help familiarize applicants with eligibility requirements, appropriate deductions, and reporting requirements.

   At recertification, if additional criteria is met, [ESAP](#)Elderly Simplified Application Project Demonstration households may be eligible for a waived interview. See [MS 605-3(D)(7)](605-3_d.md). Even if the ESAP [SNAP](#)Supplemental Nutrition Assistance Program household is eligible for a waived interview, if ineligibility exists, never close an ESAP recertification without first attempting to schedule a recertification interview.

1. Verification

   [ESAP](#)Elderly Simplified Application Project Demonstration households must comply with the verification requirements listed in [SNAP](#)Supplemental Nutrition Assistance Program [MS 601-4](../601/601-4.md). 

   The use of Data Systems and [EIS](#)Eligibility Information System Interfaces outlined in MS [601-4(A)(3)](../601/601-4.md#601_4_A_3_Information_from_Data_Systems_and_EIS_Interfaces) should be used as verification for [ESAP](#)Elderly Simplified Application Project Demonstration households, whenever possible, to verify the following factors of eligibility:

- Gross nonexempt income,
- Immigration status,
- Social security number(s),
- Identity,
- Disability, and
- Child Support (paid)

When information is not available through available data exchanges, such as income from a pension, the household must provide documentation.

1. Certification Period

   [ESAP](#)Elderly Simplified Application Project Demonstration household will be certified for 36 months.

1. Reporting and Acting on Changes During the Certification Period

   [ESAP](#)Elderly Simplified Application Project Demonstration households are subject to the same reporting requirements outlined in[SNAP](#)Supplemental Nutrition Assistance Program [MS 604-4](../604/604-3.md), however, ESAP participants are not required to submit Interim Reports.

 

Note: 

See [605-3(D)(6)](605-3_d.md) for Interim Report requirements when an [ESAP](#)Elderly Simplified Application Project Demonstration household is converted to [SNAP](#)Supplemental Nutrition Assistance Program during the certification period. 

Unlike households certified for 24 months, [ESAP](#)Elderly Simplified Application Project Demonstration households are waived from the requirement to be contacted once every 12 months. 

If a household no longer qualifies for [ESAP](#)Elderly Simplified Application Project Demonstration, they should be immediately converted back to regular [SNAP](#)Supplemental Nutrition Assistance Program. See [MS 605-3(D)(10).](605-3_d.md)

1. Recertification

   [ESAP](#)Elderly Simplified Application Project Demonstration household must complete and return the [GEN 72](#)Eligibility review application by the 15th day of the 36th month of certification in order to avoid any interruption in benefits. See [SNAP](#)Supplemental Nutrition Assistance Program [MS 604-3(B)](../604/604-2.md#604_3_B_RECERTIFICATION:_APPLICATION_FILING_DEADLINES) for additional information regarding recertification application filing deadlines. 

   The recertification interview is waived for [ESAP](#)Elderly Simplified Application Project Demonstration households unless:

- The recertification application is not provided timely and verifications are not complete;
- The household will be denied;
- Information provided is questionable, incomplete, or contradictory; or
- The household requests an interview.

1. Approval for ESAP

   When it is determined at application or recertification that a household is eligible for [ESAP](#)Elderly Simplified Application Project Demonstration, the ESAP household must be notified of their ESAP status, reporting procedures, and responsibilities.

   The caseworker will send the [SNAP](#)Supplemental Nutrition Assistance Program [ESAP](#)Elderly Simplified Application Project Demonstration Application Approved (F113) or SNAP ESAP Recertification Approved (F120) notice to these households.

1. Converting SNAP Households to ESAP

1. When it is discovered that a regular [SNAP](#)Supplemental Nutrition Assistance Program household is eligible for [ESAP](#)Elderly Simplified Application Project Demonstration at recertification, notify the household of their ESAP status, reporting procedures, and responsibilities. 

   The caseworker will send the [SNAP](#)Supplemental Nutrition Assistance Program [ESAP](#)Elderly Simplified Application Project Demonstration Application Approved (F113) notice to these households.
2. When it is discovered that a regular [SNAP](#)Supplemental Nutrition Assistance Program household is eligible for [ESAP](#)Elderly Simplified Application Project Demonstration during their certification period due to a reported change, the household will continue their current certification period and be converted to ESAP at their next recertification.

1. Converting Ineligible ESAP households to SNAP

   If an [ESAP](#)Elderly Simplified Application Project Demonstration household becomes ineligible for ESAP during their certification period, but remains eligible for [SNAP](#)Supplemental Nutrition Assistance Program, the household is removed from the ESAP demonstration and converted to a regular SNAP household. 

   The caseworker will send the [SNAP](#)Supplemental Nutrition Assistance Program [ESAP](#)Elderly Simplified Application Project Demonstration Household Converted to SNAP (F760) notice to these households. 

   [EIS](#)Eligibility Information System will automatically send the [SNAP](#)Supplemental Nutrition Assistance Program [ESAP](#)Elderly Simplified Application Project Demonstration Household Converted to SNAP (X760) when EIS knows a household is no longer eligible for ESAP.

Example: 

[EIS](#)Eligibility Information System will automatically send the X760 when a child, who is neither blind nor disabled, turns 18 and the household no longer meets [ESAP](#)Elderly Simplified Application Project Demonstration eligibility. 

The household is still entitled to the 36-month certification period but will be required to submit the Interim Report ([FSP 95](#)SNAP Interim Report) in 6 month increments for the remainder of the 36-month certification period. 

At the end of the 36-month certification period, the household will be recertified according to normal [SNAP](#)Supplemental Nutrition Assistance Program recertification requirements if they remain ineligible for [ESAP](#)Elderly Simplified Application Project Demonstration during that time.
