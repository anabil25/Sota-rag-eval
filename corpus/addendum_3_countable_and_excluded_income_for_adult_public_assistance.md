ADDENDUM 3

Countable and Excluded Income for adult public assistance

|  |  |
| --- | --- |
| Type of Income | Countable for [APA](#)Adult Public Assistance |
| Adoption payments, subsidized | Yes -- see [MS 442-3(F)](../442/442-3_unearned_income_exclusions.md#442_3_F_FOSTER_CARE_PAYMENTS) for additional details |
| American Indian/Alaska Native payments | Yes -- see [MS 442-3(A)(8)](../442/442-3_unearned_income_exclusions.md#442_3_A_INCOME_EXCLUSIONS_AUTHORIZED_BY_FEDERAL_LAWS) for additional details |
| Annuity Payments | Yes |
| Alien Sponsor income | Yes -- see [MS 460-2(B)](../460/460-2_financial_responsibility.md#460_2_B____INCOME_AND_RESOURCES_OF_ALIEN_SPONSORS) for additional details |
| Child Support | Yes -- see [MS 442-3(I)](../442/442-3_unearned_income_exclusions.md#442_3_I_CHILD_SUPPORT_PAYMENTS_FROM_AN_ABSENT_PARENT) for additional details |
| Census Income | Yes |
| Compensatory payments, court ordered | Yes |
| Contractual Income, such as monthly payments from sale of a house | Yes |
| Contract Work | Yes |
| Corporation income from partnerships, estates, and S-corporations | Yes |
| Crowdfunding | Yes -- see [MS 431-2(L)](../431/431-2_personal_property.md#431-2_L_CROWDFUNDING_) for additional details |
| Death Benefits | Yes -- see [MS 441-2(G)](../441/441-2_unearned_income.md#441_2_G_DEATH_BENEFITS) for additional details |
| Disaster and Emergency Assistance provided by states, local governments, or disaster assistance organizations | No |
| Earned Income Tax Credit (EITC) | No |
| Educational Grants, Awards, Scholarships, and Loans issued under any program funded under the Title IV of the Higher Education Act or under a Bureau of Indian Affairs student assistance program | No |
| Food Stamp Hold Harmless payment | No |
| Foster Care, including Therapeutic Foster Care and boarding home payments | No -- see [MS 442-3(F)](../442/442-3_unearned_income_exclusions.md#442_3_F_FOSTER_CARE_PAYMENTS) for additional details |
| Gambling and lottery winnings | Yes |
| Garnishments | Yes -- see [MS 443-1](../443/443-1_garnishment_or_other_withholding.md) for additional details |
| Inheritances | Yes |
| In-kind support and maintenance | No -- see [MS 440-3(A)](../440/440-3_items_that_are_not_income.md#440_3_A_THE_VALUE_OF_IN_KIND_SUPPORT_AND_MAINTENANCE) for additional details |
| Loans, including educational, personal and commercial | No |
| Military Entitlements | Yes |
| Missionary Income | Yes |
| Needs Based Assistance such as Senior Benefits | No -- see [MS 442-3(B)](../442/442-3_unearned_income_exclusions.md#442_3_B_CERTAIN_ASSISTANCE_BASED_ON_NEED) for additional details |
| Permanent Fund Dividend | No |
| Recoupments | Yes |
| Rental income, including room rental | Yes -- see [MS 441-2(M)](../441/441-2_unearned_income.md#441_2_M_RENTAL_INCOME) & [441-2(N)](../441/441-2_unearned_income.md#441_2_N_RESERVED) for additional details |
| Retirement/pension income | Yes |
| Self-employment income | Yes -- see [MS 441-1(E)](../441/441-1_earned_income.md#441_1_E_NET_EARNINGS_FROM_SELF_EMPLOYMENT) for additional details |
| Supplemental Security Income ([SSI](#)Supplemental Security Income) | Yes |
| Social Security payments, excluding [SSI](#)Supplemental Security Income | Yes |
| Spousal Support (Alimony) | Yes |
| Tax returns, federal | No |
| Title V of the Older American Act, such as MASST stipends | See [MS 442-3(A)3](../442/442-3_unearned_income_exclusions.md#442_3_A_INCOME_EXCLUSIONS_AUTHORIZED_BY_FEDERAL_LAWS) for additional details |
| Trust income | Yes |
| Unemployment benefits ([UIB](#)Unemployment Insurance Benefits) | Yes |
| Veteran Disability Payments | Yes |
| Veterans' retirement paid by the Department of Defense Finance and Accounting Services (DFAS) | Yes -- see [MS 441-2(C)](../441/441-2_unearned_income.md#441-2_C.___DEPARTMENT_OF_VETERANS_AFFAIRS_PAYMENTS) for additional details |
| Veterans Survivor benefits | Yes |
| Victims of Crime Compensation | No |
| Volunteer Payments from the Corporation for National and Community Service (CNCS), such as Senior Corps and AmeriCorps | No |
| Wages, including salaries, bonuses, commissions, tips, overtime, severance, and gratuities.  This includes an [SSI](#)Supplemental Security Income recipient's wages | Yes |
| Workers' Compensation | Yes |

This chart is not comprehensive. 

For questions regarding income types, please email [hss.dpa.policy@alaska.gov](mailto:hss.dpa.policy@alaska.gov).
