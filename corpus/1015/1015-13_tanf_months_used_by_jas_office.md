1015-13          [TANF](#)Temporary Assistance for Needy Families MONTHS USED BY [JAS](#)Jobs Automated Payments System OFFICE

This report lists each client in order of the months of [ATAP](#)Alaska Temporary Assistance Program they have used to date. This report is used as a management tool to monitor for extension consideration, see [MS 1008-2](../1008/1008-2_conducting_the_extension_review_staffing.md).

1. This report is available on [Mobius View](http://Vhttps://mobius.soa.alaska.gov/mobius/start.html)
2. There is no data dictionary to reference for this report.
3. There are no [DPA](#)Division of Public Assistance reporting requirements associated with this report.
