561           EIS  HCBHome and Community-Based WAIVER CODES

Use the eligibility code applicable to the individual if not receiving HCBHome and Community-Based waiver services (i.e. 20, 11, 24, 50, 54, etc.).EIS  will read that code and the waiver type code, then convert the code into a special waiver eligibility code and enter it on the Medical Resource (MEREMedical Reference) screen.

See [Addendum 2](../addenda/addendum_2.md) for a complete list of Medicaid Subtypes, Eligibility Codes, and Waiver Codes.
