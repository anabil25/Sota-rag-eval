ADDENDUM 2

Countable and Excluded income for Senior Benefits

|  |  |
| --- | --- |
| Type of Income | Countable for Senior Benefits |
| Adoption payments, subsidized | See [MS 330-3](../330__income/330-3_exempt_income.md) |
| American Indian/Alaska Native payments | Yes - see [MS 330-2(E)](../330__income/330-2_countable_income.md#330-2_E._ANCSA_DISTRIBUTIONS) for additional details |
| Annuity Payments | Yes |
| Alien Sponsor income | No |
| Child Support | see [MS 330-3](../330__income/330-3_exempt_income.md) |
| Census Income | Yes |
| Compensatory payments, court ordered | Yes |
| Contractual Income, such a monthly payments from sale of a house | Yes |
| Contract Work | Yes |
| Corporation Income from partnerships, estates and S-corporations | Yes |
| Crowdfunding | See [MS 330-2(F)](../330__income/330-2_countable_income.md#330-2F.____Crowdfunding_Accounts) for additional details |
| Death Benefits | Yes - see [MS 330-2(H)](../330__income/330-2_countable_income.md#330-2H_DEATH_BENEFITS) for additional details |
| Disaster and Emergency Assistance provided by states, local governments, or disaster assistance organizations | No |
| Earned Income Tax Credits (EITC) | No |
| Educational Grants, Awards, Scholarships, and Loans issued under any program funded under Title IV of the Higher Education Act or under a Bureau of Indian Affairs student assistance program | No |
| Food Stamp Hold Harmless payment | No |
| Foster Care, including Therapeutic Foster Care and boarding home payments | No |
| Gambling and lottery winnings | Yes |
| Garnishments | No - see [MS 330-3](../330__income/330-3_exempt_income.md) |
| Guardianship payments, subsidized | No |
| Interest Income | See [MS 330-2(C)](../330__income/330-2_countable_income.md#330-2_C_Interest_Income) for additional details |
| Jury Duty payments and Witness fees | Yes |
| Loans, including educational, personal and commercial | No |
| Military Entitlements | Yes |
| Missionary Income | Yes |
| Permanent Fund Dividend | No |
| Public Assistance Payments | See [MS 330-3](../330__income/330-3_exempt_income.md) for additional details |
| Recoupments | No |
| Rental Income, including room rental | Yes |
| Retirement/pension Income | Yes |
| Self-employment Income | Yes - see MS 330-2(G) for additional details |
| Supplemental Security Income ([SSI](#)Supplemental Security Income) | Yes |
| Social Security payments, excluding [SSI](#)Supplemental Security Income | Yes |
| Spousal Support (Alimony) | Yes |
| Tax returns, federal | No |
| Title V of the Older Americans Act, such as MASST | See [MS 330-2(G)](../330__income/330-2_countable_income.md#330-2G._____TITLE_V_WAGES_AND_SALARIES) for additional details |
| Trust Income | Yes |
| Unemployment Benefits (UIB) | Yes |
| Veterans' retirement paid by the Department of Defense Finance and Accounting Services (DFAS) | Yes |
| Veterans Survivor Benefits | Yes |
| Victims of Crime Compensation | No |
| Volunteer Payments from the Corporation for National and Community Service (CNCS) such as Senior Corps and Americorps | No |
| Wages, including salaries, bonuses, commissions, tips, overtime, severance, and gratuities.  This includes an [SSI](#)Supplemental Security Income recipient's wages | Yes |
| Workers' Compensation | Yes |

#

This chart is not comprehensive. For questions regarding income types, please email hss.dpa.policy@alaska.gov.
