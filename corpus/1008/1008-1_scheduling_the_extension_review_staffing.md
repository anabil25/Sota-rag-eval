1008-1          SCHEDULING THE eXTENSION REVIEW STAFFING

Early in the 57th month of [ATAP](#)Alaska Temporary Assistance Program or two months prior to the end of a current extension the [CM](#)Case Manager will discuss with the client whether they want to be considered for extended benefits.  The client can respond verbally or in writing.  If the [CM](#)Case Manager cannot discuss an extension with the client, the TA20 is mailed to the client.  If the client indicates, verbally or in writing, that they would like to be considered for an extension, the [CM](#)Case Manager will document that request and how the request was received in [CLNO](#)Client Notes.  The [CM](#)Case Manager will work with the client to schedule the Extension Review Staffing during the client's 58th month of [ATAP](#)Alaska Temporary Assistance Program.  This allows time to ensure all documentation is available to make an extension decision and for eligibility to process the Extension Review Staffing decision.  See [ATAP MS 701-1 E](http://dpaweb.hss.state.ak.us/manuals/ta/ta.htm#t=701%2F701-1_what_is_the_60-month_time_limit_.htm) for additional information.
