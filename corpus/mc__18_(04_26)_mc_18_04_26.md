Work Services Procedure Manual

[Click here to see this page in full context](#)

STATE OF ALASKA

DEPARTMENT OF HEALTH

Division of Public Assistance

TRANSMITTAL NUMBER: MC #18

MANUAL:  Work Services Procedure Manual

DATE:  April 1, 2026

We have made technical corrections and incorporated the changes outlined below.

If you have any questions, please contact the Policy and Program Development team at [hss.dpa.policy@alaska.gov](mailto:hss.dpa.policy@alaska.gov)

OVERVIEW OF CHANGES

ENTIRE MANUAL

Deleted reference to Employment Planing Interview (EPI).

Renamed Employability Planning to Employability Assessment (EA)

[MS 1001-2](../1001/1001-2_employability_assessment.md) EMPLOYABILITY ASSESSMENT

- Edited for clarity

[MS 1001-3](../1001/1001-3_conducting_the_employability_assessment.md) CONDUCTING THE EMPLOYABILITY ASSESSMENT

- Aligned with the revised TA5

[MS 1001-13](../1001/1001-13_documentation.md) DOCUMENTATION

- Replaced Employability Planning Interview with EA.
- Replaced EPI with EA.

[MS 1003-2](../1003/1003-2_work_activities.md) WORK ACTIVITIES

- Added language prohibiting the counting of work activity hours in the cannabis industry.
- Directed the client to request case closure if working in the cannabis industry.

[MS 1004-3](../1004/1004-3_what_supportive_services_cannot_be_used_for.md) WHAT SUPPORTIVE SERVICES CANNOT BE USED FOR

- Revised for clarity.
- Clarified that general home internet is not allowable supportive service.
- Added a prohibition of supportive services for clients working in the cannabis industry.

[MS 1004-7](../1004/1004-7_pass_i_childcare.md) PASS I CHILDCARE

- Created a subsection to refer to PASS I guidance.

[MS 1005](../1005/1005_job_development.md) JOB DEVELOPMENT

- Added language prohibiting placement of a client in the cannabis industry for CWE/BWE, OJT and Job Start.

[MS 1007-5](../1007/1007-5_ending_or_deleting_a_penalty.md) ENDING OR DELETING A PENALTY

- Added language to identify the intent to end the penalty and linked to ATAP MS 723-3.

[MS 1008](../1008/1008_extension_review_staffing.md) EXTENSION STAFFING

- Changed section title to Extension Staffing.
- Deleted reference to the Family Progress Review.
- Re-titled section to Extension Staffing.
- Defined Extension Staffing.

[MS 1008-1](../1008/1008-1_scheduling_the_extension_review_staffing.md) SCHEDULING THE EXTENSION STAFFING

- Deleted reference to Family Progress Review.
- Re-titled subsection.
- Provided guidance for scheduling the Extension Staffing.

[MS 1008-2](../1008/1008-2_conducting_the_extension_review_staffing.md) CONDUCTING THE EXTENSION STAFFING

- Deleted guidance for the time-limit exempt Alaska Native Village family Progress Reviews.
- Re-titled subsection.
- Provided guidance for conducting the Extension Staffing.

[MS 1008-3](../1008/1008-3_determining_the_length_of_the_extension.md) DETERMINING THE LENGTH OF THE EXTENSION

- Moved relevant content regarding evaluating the FSSP following the Family Progress Review to 1008-4.
- Re-titled subsection.
- Provided guidance for  determining the length of the extension.

[MS 1008-4](../1008/1008-4_evaluating_the_fssp_following_the_extension_review_staffing.md) EVALUATING THE FSSP FOLLOWING THE EXTENSION STAFFING

- Deleted content regarding the use of the FPR to conduct an Extension Staffing.
- Clarified the requirement to evaluate the FSSP following the Extension Staffing.

[MS 1008-5](../1008/1008-5_non-compliance_during_an_extension.md) NON-COMPLIANCE DURING AN EXTENSION

- Deleted Family Progress Review form instructions
- Added FSSP requirement and penalty information for non-compliance during an extension.

[MS 1008-6](../1008/1008-6_ending_an_extension.md) ENDING AN EXTENSION

- Added section describing the circumstances for which an extension is ended.

[MS 1015-13](../1015/1015-13_tanf_months_used_by_jas_office.md) TANF MONTHS USED BY JAS OFFICE

- Deleted reference to Family Progress Review

[MS 1018-9](../1018/1018-9_archiving_and_destruction_of_files.md) ARCHIVING AND DESTRUCTION OF FILES

- Re-titled section
- Clarified custody and responsibility for contracted Work Services files, including timeline.
- Clarified the contractor must allow electronic file storage.
- Clarified the provider must allow file access to DPA as needed.
