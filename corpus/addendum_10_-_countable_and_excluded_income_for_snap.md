ADDENDUM 10

COUNTABLE AND EXCLUDED INCOME FOR SNAP

|  |  |
| --- | --- |
| **Type of income** | **Countable for [SNAP](#)Supplemental Nutrition Assistance Program?** |
| Adoption payments, subsidized | Yes – see [MS 605-1 C](../605/605-1_c.md) for additional information. |
| American Indian/Alaska Native payments | See [Addendum 5](addendum_5.md) for additional information. |
| Annuity Payments | Yes – see [MS 602-3 B](../602/602-3_b.md) for additional information. |
| Alien Sponsor income | Yes - see [MS 605-2 A](../605/605-2_a.md) for additional information. |
| Child Support | Yes – see [MS 602-3 B(3)](../602/602-3_b.md#602_3_B_3__Child_Support_Payments) for additional information. |
| Census Income from 2020 Census | No |
| Compensatory payments, court ordered | Yes |
| Contractual Income, such as monthly payments from sale of a house | Yes |
| Contract Work | Yes |
| Corporation For National and Community Service (CNCS) payments, which include AmeriCorps, VISTAVolunteers in Service to America, National and Civilian Corp, Foster Grandparents, the Senior Companion Program, and Retired and Senior Volunteer Corps (RSVP) | No – see [MS 602-3 D](../602/602-3_d._exempt_income.md) for additional information. |
| Corporation income from partnerships, estates, and S-corporations | Yes |
| Crowdfunding | See [MS 602-2 A](../602/602-2_a.md#602_2_A_1__Liquid_Resources) for additional information. |
| Disaster and Emergency Assistance provided by states, local governments, or disaster assistance organizations | No |
| Earned Income Tax Credit (EITC) | No |
| Educational Grants, Awards, Scholarships, and Loans issued under any program funded under the Title IV of the Higher Education Act or under a Bureau of Indian Affairs student assistance program | No |
| Foster Care, including Therapeutic Foster Care and boarding home payments | Dependent on nature of foster care payments, and household living expenses – See [MS 605-1 C](../605/605-1_c.md) for additional information. |
| Gambling and lottery winnings | See [MS 605-1 C](../605/605-1_c.md) for additional information. |
| Garnishments | See [MS 602-3 B](../602/602-3_b.md#602_3_B_7__Other_Sources_of_Countable_Unearned_Income) for additional information. |
| Gifts | No – see [MS 602-3 D 9](../602/602-3_d._exempt_income.md) for additional information |
| Guardianship payments, subsidized | See [MS 605-1 C](../605/605-1_c.md) for additional information. |
| Inheritances | No |
| In-kind income | No - See [602-3 D](../602/602-3_d._exempt_income.md) for additional information. |
| Interim Assistance | Yes |
| Jury Duty payments and Witness fees | Yes |
| Jury Duty Reimbursements and per diem | No |
| Loans, including educational, personal, and commercial | No |
| Lump sum payments, non-recurring | No |
| MAAST (Mature Alaskans Seeking Training Skills) | No – See MS [602-3 D](../602/602-3_d._exempt_income.md) |
| Military Entitlements | Yes – see [MS 605-2 E](../605/605-2_e.__military_pay.md) for additional information. |
| Missionary Income | Yes |
| PASS I, II, III Childcare | No |
| Payee Arrangements | See [MS 602-3 B](../602/602-3_b.md#602_3_B_4__Third_Party_Payee)  for additional information. |
| Per Diem payment from employment | No, See [Addendum 3](addendum_3.md) for Military personnel |
| Permanent Fund Dividend | See [MS 605-7](../605/605-7_permanent_fund_processing.md) for additional information. |
| Punitive Payments, court ordered | Yes |
| Public Assistance Payments | Yes |
| Refugee Assistance, paid one time or monthly | Yes see [MS 602-3 B](../602/602-3_b.md#602_3_B_7__Other_Sources_of_Countable_Unearned_Income) for additional information |
| Recoupments | No, see [MS 602-3 D](../602/602-3_d._exempt_income.md) for additional information |
| Reimbursements | No – see [MS 602-3 D](../602/602-3_d._exempt_income.md) for additional information |
| Relocation payments | No |
| Rental income, including room rental | Yes |
| Retirement/pension income | Yes – see [MS 602-3 B](../602/602-3_b.md#602_3_B_7__Other_Sources_of_Countable_Unearned_Income) for additional information. |
| Self-employment income | Yes - see [MS 605-2 D](../605/605-2_d.md) for additional information. |
| Supplemental Security Income (SSISupplemental Security Income) | Yes |
| Social Security payments, excluding SSISupplemental Security Income. | Yes - see [MS 602-3 B](../602/602-3_b.md#602_3_B_2__SSI_SSA_VA_Income) for additional information. |
| Spousal Support (Alimony) | See [MS 603-1](../603/603-1.md#603_1_D_IRREGULAR_INCOME) and [Addendum 3](addendum_3.md) |
| Tax returns, federal, | No |
| Trust income | Yes, see [MS 602-3 B](../602/602-3_b.md#602_3_B_7__Other_Sources_of_Countable_Unearned_Income) for additional information. |
| Unemployment benefits (UIBUnemployment Insurance Benefit) | Yes |
| Vendor payments | See [MS 602-3 B](../602/602-3_b.md) for additional information. |
| Veterans retirement paid by the Department of Defense Finance and Accounting Services (DFAS) | Yes |
| Veterans Survivor benefits | Yes |
| Victims of Crime Compensation | No |
| Wages, including salaries, bonuses, commissions, tips, overtime, severance, and gratuities. This includes an SSISupplemental Security Income recipient’s wages. | Yes |
| Workers' Compensation | Yes |
| Work Study | No |

This chart is not comprehensive. For questions regarding income types, please email [hss.dpa.policy@alaska.gov](mailto:hss.dpa.policy@alaska.gov).
