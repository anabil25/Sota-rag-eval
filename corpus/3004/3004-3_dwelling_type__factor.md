3004-3      DWELLING TYPE AND FACTOR

The number of bedrooms, square footage, or type of structure is an important and necessary factor in determining an applicant's eligibility and grant amount. Please see the following chart that outlines the various types of dwellings the program serves and their corresponding dwelling factor (multiplier) used to determine the applicant's benefit amount.

|  |  |
| --- | --- |
| Type of Dwelling | Dwelling Factor |
| Mobile home or boat with heated living space of 980 square feet or more | 1.4 |
| Recreational vehicle, tent, truck, van, car or pickup camper | 0.4 |
| One-room dwelling such as a studio apartment, hotel, or boarding home | 0.15 |
| Boat with heated living space less than 980 square feet | 0.35 |
| One-bedroom dwelling or a one-room house or cabin without bedrooms | 0.55 |
| Three-or-more-bedroom single family, duplex, or triplex home | 1.3 |
| Two-bedroom unit in an apartment building of four or more attached units, or resides in a mobile home with less than 980 square feet of heating living space, regardless of the number of bedrooms | 0.7 |
| Two-bedroom single family home, duplex, or triplex home | 1.0 |
