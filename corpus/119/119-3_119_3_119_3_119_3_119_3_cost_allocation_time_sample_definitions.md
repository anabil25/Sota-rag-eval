119-3          COST ALLOCATION TIME SAMPLE DEFINITIONS

The definitions listed in this section correspond to the code types on the [RMS](#)Random Moment Sample observation form. The caseworker must respond based on the activity that is currently being performed at the time of the [RMS](#)Random Moment Sample sampling.

119-3 A.      PROGRAM CODES AND DEFINITIONS

For each moment of RMSRandom Moment Sample activity recorded, the worker must carefully review the list of activities and select the activity that most appropriately reflects what the caseworker is working on at the randomly chosen moment.

If at the selected moment the worker is processing an application or case with two or more programs or combination of programs, and is unsure which program code to allocate the activity, use the following guidelines:

- If the activity at the "moment" is discrete to one program, then the selection of the corresponding specific program is the correct choice.
- If the worker is asking information or gathering information for completing an eligibility redetermination for a specific program area, that program should be selected on the RMSRandom Moment Sample Observation form. Otherwise, the appropriate code is used for the combination of programs.

For example, the worker is processing an application for Adult Public Assistance, Supplemental Nutrition Assistance Program, and Medicaid. If at the time study moment the worker is authorizing the Medicaid Program, then the specific "APA/Medicaid" program should be selected.

For this sample application, if the worker is verifying income belonging to a household member who will be included only in the Supplemental Nutrition Assistance Program benefit, select the specific "Supplemental Nutrition Assistance Program" option. Or, if the worker is entering on EISEligibility Information System the birthdate for the head of household, it is of common benefit to the SNAPSupplemental Nutrition Assistance Program/APAAdult Public Assistance/MEMedicaid programs then the multi-program combination selection should be used.

The codes in this section are not case-specific. Do not use a code in this series if there is a case specific activity code in [MS 119-3 A](#119-3_A._PROGRAM_CODES_AND_DEFINITIONS) that better describes the activity.
