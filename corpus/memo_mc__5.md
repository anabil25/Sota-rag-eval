MEMORANDUM

DATE:          December 20, 2002

TO:               Adult Public Assistance Manual Holders

FROM:         Ellie Fitzjarrald, Chief

                      Policy and Program Development Team

SUBJECT: Adult Public Assistance Manual Change #5

Effective January 1, 2003, a 1.4% Cost of Living Allowance (COLA ) increase will be granted to recipients of Social Security and SSISupplemental Security Income. This COLACost of Living Allowance increase also affects the APAAdult Public Assistance program. Beginning with the benefit month of January 2003, this manual change implements new APAAdult Public Assistance need and maximum payment standards, increases the income limit for substantial gainful activity, and raises the earned income exclusion of a blind or disabled student child.

This manual change also eliminates the prohibition against DPADivision of Public Assistance employees acting as payees, and clarifies policy pertaining to SSISupplemental Security Income alien status requirements and property agreements. It also makes other minor technical corrections.

Implementation Dates:  The 2003 COLACost of Living Allowance increases are effective January 1, 2003. All other changes are effective upon receipt of this manual change.

The explanation of the manual changes and the affected manual sections are described below. If you have any questions, please contact any member of the Policy and Program Development Team at 465-3347 or e-mail us at dpapolicy@health.state.ak.us.

|  |  |
| --- | --- |
| Old Policy | New Policy |
| Section [421-10](../../421/421-7_ssi_alien_status_requirements.md), SSISupplemental Security Income Alien Status Requirements | |
| Describes the alien status requirements of the SSISupplemental Security Income program. | Clarifies that aliens who meet APAAdult Public Assistance alien status requirements may still qualify for APAAdult Public Assistance even if they do not meet the more restrictive alien status requirements of the SSISupplemental Security Income program. |
| Section [425-1](../../425/425-1_definition_of_blindness_disability.md), Definition of Blindness/Disability | |
| Defines substantial gainful activity (SGA ) as an individual’s own average monthly earned income of $780. | Raises the SGASubstantial Gainful Activity amount to $780 per month. |
| Section [431-2I](../../431/431-2_personal_property.md#431_2_I_PROMISSORY_NOTES__LOANS__AND_PROPERTY_AGREEMENTS), Promissory Notes, Loans, and property Agreements | |
| Describes how promissory notes, loans, and property agreements are treated in determining APAAdult Public Assistance eligibility. | Clarifies that the resource value of a promissory note, loan, or property agreement is its fair market value when the fair market value is less than the outstanding principal balance. |
| Section [442-2D](../../442/442-2_earned_income_exclusions.md#442_2_D_STUDENT_EARNED_INCOME_EXCLUSION), Earned Income Exclusion of a Blind or Disabled Student Child | |
| Provides for an earned income exclusion against the earnings of a blind or disabled student child of up to $1,320 per month, but not more than $5,340 per calendar year. | Raises the earned income exclusion to $1,340 per month, and $5,410 per calendar year. |
| Section 451-3, APAAdult Public Assistance Need Standards | |
| Provides APAAdult Public Assistance need standards for 2002. | Updates the APAAdult Public Assistance Need Standards for 2003. |
| Section [452-7B](../../452/452-7_to_whom_payment_is_made.md#452_7_B____PAYMENT_TO_A_THIRD_PARTY_PAYEE), Payment to a Third Party | |
| Describes the policies for providing APAAdult Public Assistance payments to a payee. | Eliminates the prohibition against DPADivision of Public Assistance employees acting as a payee. |
| Section 453, Maximum Payment Standards | |
| Gives APAAdult Public Assistance and SSISupplemental Security Income maximum payment standards for 2002. | Updates the APAAdult Public Assistance and SSISupplemental Security Income maximum payment standards for 2003. |
