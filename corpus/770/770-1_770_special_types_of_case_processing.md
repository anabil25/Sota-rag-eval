770-1          [SSI](#)Supplemental Security Income CHILD CASES

The Social Security Administration (SSA) may issue monthly Supplemental Security Income (SSI) payments to blind or disabled children under the age of 18. Regardless of the [ATAP](#)Alaska Temporary Assistance Program policies regarding adult [SSI](#)Supplemental Security Income recipients, these [SSI](#)Supplemental Security Income children may be included in the [ATAP](#)Alaska Temporary Assistance Program assistance unit if the caretaker relative requests it. In situations where the only dependent child in the assistance unit is an [SSI](#)Supplemental Security Income recipient, eligibility for [ATAP](#)Alaska Temporary Assistance Program does not exist if the caretaker relative chooses not to include that child in the assistance unit.

The monthly countable [SSI](#)Supplemental Security Income income of a child receiving [SSI](#)Supplemental Security Income is equal to the [ATAP](#)Alaska Temporary Assistance Program need standard increment for an additional child, regardless of size of the assistance unit or the actual amount of [SSI](#)Supplemental Security Income received. See [Addendum 2](../addenda/addendum_2_-_program_standards.md) for the need standard increment for an additional child. Any [RSDI](#)Retirement Survivors Disability Insurance benefits that child is eligible for is also countable income if that child is included in the [TA](#)Temporary Assistance household.

The intent of this provision is to allow the parent or caretaker relative to receive the services [ATAP](#)Alaska Temporary Assistance Program provides without including the needs of the [SSI](#)Supplemental Security Income child in the assistance unit. Removing the [SSI](#)Supplemental Security Income child's needs from the [ATAP](#)Alaska Temporary Assistance Program benefit payment prevents [SSA](#)Social Security Administration from deeming [ATAP](#)Alaska Temporary Assistance Program income toward the [SSI](#)Supplemental Security Income child's [SSI](#)Supplemental Security Income benefit payment.

Note:

[SSA](#)Social Security Administration may or may not be deeming [ATAP](#)Alaska Temporary Assistance Program income towards the child's [SSI](#)Supplemental Security Income benefit. If it appears that the child's [SSI](#)Supplemental Security Income is being reduced due to [ATAP](#)Alaska Temporary Assistance Program / [TANF](#)Temporary Assistance to Needy Families income ‒ send the [SSA](#)Social Security Administration an [APA 3](#)SSA Fax Information Request Form form to inform them that the [SSI](#)Supplemental Security Income child's needs are not included in the [ATAP](#)Alaska Temporary Assistance Program household.
