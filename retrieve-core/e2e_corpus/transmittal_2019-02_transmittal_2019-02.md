Administrative Procedures Manual

[Click here to see this page in full context](#)

STATE OF ALASKA

Department of Health and Social Services

Division of Public Assistance

TRANSMITTAL NUMBER: 2019-02

MANUAL: Administrative Procedures Manual

DATE: September 3, 2019

Several sections of the Administrative Procedures Manual have been revised.

If you have any questions, please contact Field Services at [dpafield@alaska.gov](mailto:dpafield@alaska.gov) or the Policy & Program Development Team at [dpapolicy@alaska.gov](mailto:dpapolicy@alaska.gov).

OVERVIEW OF CHANGES

[MS 100-3(H)](../100/100-3_confidentiality.md#100-3_H._REQUESTS_FOR_INFORMATION_FROM_OTHER_AGENCIES)      REQUESTS FOR INFORMATION FROM OTHER AGENCIES

- Added Child Care Assistance Program Grantees to the list of agencies that DPA can release information without written consent from the client since a MOA is in place

[MS 102-3](../102/102-3_telephone_interviews.md)      TELEPHONE INTERVIEWS

- Referred staff to CANO Desk Guide posted on the DPAweb as Appendix 109 A has been deleted

[MS 103-1(B)](../103/103-1_electronic_document_management.md#103-1_B._GENERAL_RULES_FOR_CREATING_CASE_FILE_FOLDERS)      ORGANIZATION OF THE SIX-PART FILE FOLDER

- Replaced reference to CSED 1603 form with Child Support Information form (GEN 53) throughout this section

[MS 103-1(C)](../103/103-1_electronic_document_management.md#103-1_C._ORGANIZATION_OF_THE_SIX-PART_FILE_FOLDER)      ORGANIZATION OF THE TWO-PART FILE FOLDER

- Replaced reference to CSED 1603 form with Child Support Information form (GEN 53) throughout this section

[MS 103-1(F)](../103/103-1_electronic_document_management.md#103-1_F._PURGING_CASE_FILES)      CREATING CASE FILE VOLUMES

- Replaced reference to CSED 1603 form with Child Support Information form (GEN 53) throughout this section

[MS 103-2](../103/103-2_identifiable_applications.md)      REQUEST FOR SERVICE

- Replaced reference to Family Medicaid with MAGI Medicaid throughout this section
- Added note clarifying that an interview is not required for MAGI Medicaid
- Revised request for service example for clarity
- Referred staff to HowWe Guide Addendum 1 (Statewide Request for Service Job Aid) for additional information when request for service is received
- Clarified that MAGI Medicaid is not registered as a request for service

[MS 104-2](../104/104-2_recertification_and_review.md)      RECERTIFICATION AND REVIEW

- Replace reference to Family Medicaid with MAGI Medicaid throughout this section

[MS 105-1(B)](../105/105-1_verification_of_information.md#105-1_B._MANDATORY_VERIFICATION)      MANDATORY VERIFICATION

- Removed the verification matrix from the Admin Procedure Manual as staff will used the verification matrix in the HowWe Guide

[MS 109-6](../109/109-6_eis_aries_cano_guidance.md)      EIS/ARIES CANO GUIDANCE

- Referred staff to CANO Desk Guide posted on the DPAweb as Appendix 109 A has been deleted
- Section title changed from CANO Desk Guide (in Table of Contents) and CANO Format (in Section page) to EIS/ARIES CANO Guidance

[MS 121-13](../121/121-13_closing_cases_or_ending_payments.md)      CLOSING CASES OR ENDING PAYMENTS

- Removed reference to Family Medicaid

[MS 123-2](../123/123-2_potential_unemployment_insurance_benefit_eligibility.md)      POTENTIAL UNEMPLOYMENT INSURANCE BENEFIT ELIGIBILITY

- Replaced reference to Family Medicaid with MAGI Medicaid

[MS 124-1(B)](../124/124-1_child_or_youth_taken_into_ocs_custody.md#124-1_B._YOUTH_OVER_AGE_18_RE-ENTERS_OCS_CUSTODY)      YOUTH OVER AGE 18 RE-ENTERS OCS CUSTODY

- Replaced reference to Family Medicaid with MAGI Medicaid

[ADDENDUM 109 A](../addenda/addendum_109_a_-_reserved.md)      CANO FORMAT

- Removed addendum as staff will use CANO Desk Guide posted on the DPAweb
