Administrative Procedures Manual

[Click here to see this page in full context](#)

STATE OF ALASKA

Department of Health and Social Services

Division of Public Assistance

TRANSMITTAL NUMBER: 2021-03

MANUAL: Administrative Procedures Manual

DATE: December 1, 2021

Several sections of the Administrative Procedures Manual have been revised.

If you have questions, please contact Division Operations Support Team at [hss.dpadost@alaska.gov](mailto:hss.dpadost@alaska.gov) or the Policy and Program Development Team at [hss.dpa.policy@alaska.gov](mailto:hss.dpa.policy@alaska.gov).

OVERVIEW OF CHANGES

[MS 100-11 F](../100/100-11_right_to_file_a_complaint.md#100-11_F._PROCEDURE_FOR_SUPERVISORS_AND_MANAGERS_FOR_HANDLING_COMPLAINTS)      PROCEDURES FOR SUPERVISORS AND MANAGERS FOR HANDLING COMPLAINTS

- Updated references from regional manager to Field Services Manager as the job title has changed

[MS 103-1](../103/103-1_electronic_document_management.md)      ELECTRONIC DOCUMENT MANAGEMENT SYSTEM

- Renamed section from "The Case File Folder" to "Electronic Document Management System"
- Removed process related to paper file folder as it is no longer valid since all documents are stored in the ILINX. Specific subsections were removed: 103-1B General Rules for Creating Case File Folders; 103-1D Organization of the Manila Folder; 103-1E Purging Case Files; 103-1F Creating Case File Volumes; and 103-1G General Rules for Creating Temporary Files
- Sections 103-1A and 103-1C are renumbered to 103-1 and 103-1A

[MS 103-2](../103/103-2_identifiable_applications.md)      IDENTIFIABLE APPLICATIONS

- Renamed section from "Request for Service" to "Identifiable Applications"
- Updated process for handling identifiable applications

[MS 105-10](../105/105-10_social_security_enumeration.md)      SOCIAL SECURITY ENUMERATION

- Added note that legal names should be used in the eligibility systems. The legal name is what is used in Social Security Administration records so if an individual's social security card has not been corrected to reflect their current name, the applicant or recipient must correct their social security card

[MS 106-5](../106/106-5_instant_eligibility_verification_system_(ievs).md)      INSTANT ELIGIBILITY VERIFICATION SYSTEM (IEVS)

- Added new IEVS sources
- Clarified that APPRISS provides national incarceration data

[MS 115-8 E](../115/115-8_bureau_of_indian_affairs_general_assistance.md#115-8_E._BIA___DPA_INFORMATION_EXCHANGE)      BIA / DPA INFORMATION EXCHANGE

- Removed references to regional manager as this is no longer a valid position

[MS 117-4 A](../117/117-4_pre_hearing_conference.md#117-4_A._SCHEDULING_THE_PRE-HEARING_CONFERENCE)      SCHEDULING THE PRE-HEARING CONFERENCE

- Replaced reference of regional manager to eligibility manager since the position regional manager has been renamed to eligibility manager and the old tile is no longer valid

[MS 117-4 B](../117/117-4_pre_hearing_conference.md#117-4_B._RESPONSIBILITIES_OF_THE_SUPERVISOR_OR_ELIGIBILITY_MANAGER)      RESPONSIBILITIES OF THE SUPERVISOR OR ELIGIBILITY MANAGER

- Renamed title section from "Responsibilities of the Supervisor, Site Manager, or Regional Manager" to "Responsibilities of the Supervisor or Eligibility Manager" since regional manager position is no longer valid and has been renamed to eligibility manager
- Updated where staff forwards fair hearing request from dpa.csqa@alaska.gov to I-Sight

[MS 117-4 C](../117/117-4_pre_hearing_conference.md#117-4_C._PRE-HEARING_CONFERENCE_FOR_INTERIM_ASSISTANCE_DENIALS_BASED_ON_MEDICAL_REVIEW)      PRE-HEARING CONFERENCE FOR INTERIM ASSISTANCE DENIALS BASED ON MEDICAL REVIEW

- Replaced reference of regional manger to eligibility manager since the position regional manager has been renamed and the old title is no longer valid

[MS 117-5](../117/117-5_dpa_hearing_representative.md)      DPA HEARING REPRESENTATIVE

- Replaced reference of regional manager to eligibility manager since the position regional manager has been renamed to eligibility manager and the old title is no longer valid

[MS 117-10](../117/117-10_conducting_the_fair_hearing.md)      CONDUCTING THE FAIR HEARING

- Clarified that hearings are conducted at the Office of Administrative Hearings in Anchorage

[MS 117-12 B](../117/117-12_the_fair_hearing_decision.md#117-12_B._DENIAL_OR_DISMISSAL_OF_A_FAIR_HEARING)      DENIAL OR DISMISSAL OF A FAIR HEARING

- Replaced reference of regional manager to eligibility manager since the position regional manager has been renamed and the old title is no longer valid

[MS 121-2](../121/121-2_who_may_authorize_payments.md)      WHO MAY AUTHORIZE PAYMENTS

- Clarified who may authorized payments through EIS

[MS 121-3 D](../121/121-3_to_whom_payments_will_be_made.md#121-3_D._CARE_OF__A_DPA_OFFICE)      "CARE OF" A DPA OFFICE

- Replaced references of regional manager to eligibility as regional manager is no longer a valid position

[MS 121-16](../121/121-16_canceling_outstanding_payments.md)      CANCELING OUTSTANDING PAYMENTS

- Added new subsection regarding canceling outstanding APA and SB payments issued after a client's date of death
