| ADDENDUM 109 D | PREHEARING CONFERENCE CANO |

|  |  |
| --- | --- |
| SUBJECT | ELEMENTS |
| RECORD TITLE | Prehearing Conference |

|  |  |
| --- | --- |
| REQUEST DATE: | 1. List date client requested fair hearing. 2. List date of pre-hearing conference. 3. List which program(s) fair hearing is requested for. |
| ISSUE: | Summarize the issue chronologically. |
| FACTS OF CASE: | In narrative form, address each of the following:  1. Why was an adverse action taken? 2. Cite all applicable manual sections. 3. Any policy clarification on issue? 4. Was correct notice sent? 5. Conciliation attempted by caseworker? 6. Was policy explained to client? 7. Does good cause apply? Why/Why not? 8. Were continued benefits addressed? |
| CLIENT CONTACT: | 1. Was conference by phone or in person? 2. If unable to contact client by phone, was notice sent to    client? 3. Was client advised of their right to legal representation    (i.e. AK Legal Services, Disability of Law Center, etc.)? |
| STATUS: | 1. Was part of or all issue(s) resolved? 2. Will client receive continued benefits? 3. State date fair hearing request forwarded. |
